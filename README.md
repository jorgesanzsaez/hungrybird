# 👷‍♂️ How to run

Clone repository

```bash
git clone https://github.com/r3nanp/flappy-bird.git
```

Then, install the dependencies:

```bash
yarn
```

or

```bash
npm install
```

Run in your browser:

```bash
yarn dev
```
